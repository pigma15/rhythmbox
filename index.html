Welcome to Rhythmbox

transport buttons

Pressing button activates play mode and step sequencers of each track start to run.
Pressing button deactivates play mode and returns their active step to first position (for info about "active step" look below).
Bonus tip: In case of any malfuctions resetting machine by simply clicking and then clicking afterwards might help.

active step

As long as machine is in play mode step sequencers of each track will be running in loops.
Active step (or current position of loop) is shown with orange circular light.

triggers

Steps can be turned on and turned off by simply clicking on them. Sky blue light will appear as soon as step is turned on. Turning off the step will make light disappear.
Step which has been turned on will trigger sound as it becomes active (with orange light around).
However, it is possible to change probability of sound to be triggered by turning on/off rectangular button. When probability button is turned(illuminated) it gives 50% chance for sound to be triggered.

tempo

Tempo adjusts speed of step sequencers. Step sequencers start to run faster with higher values.
Tempo can be adjusted with arrow buttons on the right of tempo srceen.

tonality

Sounds of tertia and septima tracks can be changed (from minor third to major third and from minor seventh to major seventh) by editing value on tonality screen.
Tonality can be changed only while play mode is not active.

step count

Amount of steps for each track can be changed by editing values with arrow buttons on the right of step count screen.
Time which takes one cycle of sequencer's loop to complete does not change by adding/removing steps. However, it changes how much time it takes to jump from one step to another.

tempo multiplier

It is possible to change speed of sequencer on each track seperately by editing values in tempo multiplier section.
Individually edited tempo depends on values of global tempo. For example if global tempo is 100 then track with tempo multiplier's value of 1/2 will run at speed of 50 and if multiplier is set to 3/2 it will run as if tempo were 150.
